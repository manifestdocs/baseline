Core Principles

Three pillars. One foundation.

01

The Type Is the Spec

Types encode correctness using refinements. A Port is not just an Int -- it is an integer between 1 and 65535, proven at compile time. The type carries the truth.

02

Effects Are Data

Side effects are explicit capabilities tracked in type signatures. Console, Http, Fs -- each is declared, each is enforced. The boundary is the type.

03

LLM-Native

Designed for machine generation with unambiguous syntax. AI agents are sandboxed by the effect system -- they can only access what you explicitly grant.
