A Language That Proves What Is Real

Baseline

Strongly typed. Effect-tracked. Verified at compile time. A programming language where the type system is the specification.

Certainty, engineered.

Core Principles

01

The Type Is the Spec

"The structure declares its purpose. Nothing is hidden."

Types encode correctness using refinements. A Port is not just an Int -- it is an integer between 1 and 65535, verified at compile time. No defensive coding needed.

02

Effects Are Data

"Every boundary is intentional. Every capability, granted."

Side effects are explicit capabilities tracked in type signatures. A function that reads files declares {Fs}. A pure function declares nothing. The compiler enforces boundaries.

03

LLM-Native

"Built for the machines that build for us."

Designed for machine generation with unambiguous syntax. AI agents are sandboxed by the effect system -- they can only access capabilities you grant. Verification is automatic.

Built for Correctness

Every feature serves verification. Nothing is incidental.

Refinement Types

Encode value constraints directly in types. Prove correctness at compile time.

Algebraic Effects

Track all side effects in type signatures. Console, Http, Fs -- explicit capabilities.

Pattern Matching

Exhaustive match expressions for algebraic data types. No missed cases.

Pipe Operator

Compose transformations with |>. Data flows left to right, clear and readable.

ML-Style Syntax

Clean function signatures: name : Sig = body. Types read naturally.

Sum Types

Discriminated unions model domain variants. Result, Option, and your own types.

Records

Structural typing for data. Named fields, dot access, type-safe construction.

Security Sandbox

The effect system doubles as a capability sandbox for AI agents and plugins.
