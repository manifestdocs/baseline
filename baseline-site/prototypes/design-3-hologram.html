Core Principles

Three Axioms of Verification

T

The Type Is the Spec

Types encode correctness using refinements. A Port is not just an Int -- it is proven to be within valid bounds at compile time. The type is the truth.

E

Effects Are Data

Side effects are explicit capabilities tracked in type signatures. Console, Http, Fs -- each declared, each enforced. Pure by default.

L

LLM-Native

Designed for machine generation with unambiguous syntax. The effect system sandboxes AI agents -- they access only what you grant.
