Interrogator

What do your types tell you?

Recite your baseline. What is a Port? What is truth encoded in syntax?

Response
type Port = Int where 1 <= self <= 65535

listen : Port -> {Net} Server
listen = |port|
  // port is GUARANTEED to be 1-65535 here
  // No defensive coding needed
  ...

The type is the spec. Refinement types encode value constraints directly. A Port is not an Int -- it is an Int proven to be within bounds. The compiler verifies. No runtime checks. No exceptions. No doubt.

greet : String -> String
greet = |name| "Hello, ${name}!"

main! : () -> {Console} ()

LLM-native. Unambiguous syntax with no overloaded semantics. Every function signature is its contract. Every effect is its permission. Machines generate the code. The compiler verifies it. Humans approve.

Interrogator

How do you compose?

Show me data flowing. Transformation. Reduction. The pipeline.

Response
let chained = nums
  |> |xs| List.map(xs, |x| x * 2)
  |> |xs| List.filter(xs, is_even)
  |> |xs| List.fold(xs, 0, |acc, x| acc + x)

Pipe operators compose transformations left to right. Each step is a function. The data flows. Map, filter, fold -- the eternal sequence.

Three Constants

The axioms of the language. Recite.

1

The type is the spec

Types encode correctness using refinements. A type is not documentation -- it is proof. The compiler enforces what humans intend.

2

Effects are data

Side effects are explicit capabilities tracked in signatures. Pure functions have no effects. Effectful functions declare them. The boundary is the type.

3

LLM-native

Designed for machine generation with unambiguous syntax. AI agents are sandboxed by effects -- they can only access what you grant. Verification is automatic.

Capabilities

Refinement Types
Encode value constraints in types. Prove correctness at compile time.
Algebraic Effects
Track side effects in type signatures. Console, Http, Fs -- explicit capabilities.
Pattern Matching
Exhaustive match for sum types. No missed cases. No runtime surprises.
Pipe Operator
Compose with |>. Data flows left to right, clear and predictable.
ML-Style Syntax
name : Sig = body. Functions declare their contracts up front.
Sum Types
Discriminated unions for domain modeling. Result, Option, and custom variants.
Records
Structural data types. Named fields, dot access, type-safe construction.
Security Sandbox
Effect system as capability sandbox. AI agents, plugins -- controlled access.
